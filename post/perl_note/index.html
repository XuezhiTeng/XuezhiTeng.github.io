Lecture #1 Hello World

use strict;
#It appears at the head of any perl program. It is used for variable declaration. (kinda force you to declare a variable before using it and prevent you from misspelling it)
use warnings;
#It also appears at the head of a perl program and is acting with a similar function as the "use strict" statement. (there will be a warning if you violate any restrction)

sub main {
    print " Hello World";  #这里可以加上括号变成("Hello World")
}

main ();

上面的main可以叫其他别的名字(但是上下要保持一致)，而sub代表这是一个sub-routine

Lecture #2 Downloading Text and Images with Perl

use LWP::Simple;
#simple procedural interface to LWP, 可以暂时简单理解为一个包含interface的perl package

use strict;
use warnings;
use LWP::Simple;

sub main {
	print "Downloading...\n";
	print get("http://users.ece.utexas.edu/~patt/"); #Note1

	getstore("http://users.ece.utexas.edu/~patt/", "yalepatt.html");#Note2


	my $code = getstore('http://users.ece.utexas.edu/~patt/ph.bright.jpg',
    "yalepatt.png");#Note3

	if ($code == 200){ #200 is success code
		print "Succeed\n";
	}
	else {
		print "Fail\n";
	}
	
	print "Finished\n";
}

main();

Note:

  1. 这里是将上述网址的html代码获取(get)然后再打印出来(print)
  2. 这里是将上述网址直接保存为一个html文件，逗号后面跟的是保存的名字。值得注意的是，如果该网页有图片的话，是不会显示在这个html打开之后的网页的（除非这个图片就保存在你的本地！）
  3. 这里似乎单引号或者双引号都可以,在perl里，双引号代表里面的字符有特殊含义，而单引号则代表只是单纯的字符而已。这里是将网页上的某张图片保存在本地的方法，图片的链接是在网页上右键获得。getstore这个method是有返回值的，在上面的代码中使用my的方式声明了一个变量$code来存储这个返回值
